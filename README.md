# Hello GraphQL
This project is a GraphQL example built with React and based on [Create React App](https://github.com/facebookincubator/create-react-app).

## Prerequisites
- [Git](https://git-scm.com/book/en/v2/Getting-Started-Installing-Git)
- Node.js v6.9+ and NPM v3.x â€“ we recommend using [NVM (Linux/Mac)](https://github.com/creationix/nvm) or [NVM-Windows (Windows)](https://github.com/coreybutler/nvm-windows)

## Getting Started
To get started run the commands below.

```bash
$ git clone https://github.com/onehungrymind/hello-graphql-react.git
$ cd hello-graphql-react
$ npm install
$ npm start
```

Then navigate to [http://localhost:3000](http://localhost:3000) in your browser.
